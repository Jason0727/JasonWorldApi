# 采购合同

## 一、添加新合同
### 流程:
### 1、点击"添加"按钮，弹出"添加新合同"框
### 2、选择客户类别，发货地区，填写合同编号，合同名称以及银行合同编号
### 3、点击"确认"按钮，执行保存
